# 易象 — 现代命理应用设计文档

> 2026-06-02 | 版本 v0.1

## 一、产品概述

**「易象」（Yixiang）** 是一个基于中国传统命理学（易经、八字、五行等）的现代算命应用。先上线网页端，后续扩展到微信小程序。

### 核心定位

- 面向传统文化爱好者，内容专业可信
- 界面现代简洁，区别于传统老旧算命网站
- 基础功能免费引流，深度分析付费变现

## 三、商业模式

### 收入模型

算命类产品使用频次低、偶发性强，不适合订阅制。采用：

- **免费层**：日常运势、八字基础排盘、每日1次易经占卜、黄历查询
- **永久VIP（买断制）**：一次付费，永久解锁全部功能和深度内容
  - 建议定价：¥88-128
- **单次报告（按需）**：针对只想看某一个深度报告的临时用户
  - AI深度报告 ¥3-9/次

### 内容策略

- 免费内容：预置模板规则拼装（零边际成本），信息完整但不深入
- VIP/付费内容：**DeepSeek API** 实时生成个性化深度解读
  - 成本极低：约 ¥1/百万 token，一份深度报告成本不到 1 分钱

## 四、视觉设计

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面底色 | `#FAF6EE` | 宣纸暖白，温暖自然 |
| 主文字 | `#2C2416` | 深棕，柔和有力 |
| 次要文字 | `#7A6E5E` | 中棕 |
| 辅助文字 | `#B8A88A` / `#A6947A` | 浅棕 |
| 卡片背景 | `#FFFFFF` | 纯白，在暖底上凸显 |
| 分割线/灰底 | `#E8E0D4` | 极浅棕 |
| 强调色 | 无额外强调色 | 纯靠深棕+暖白对比 |

### 设计原则

- 零装饰元素：不加水墨纹理、祥云图案、八卦底纹
- 排版和留白驱动视觉层次
- 图标：Unicode 符号 + 深棕细线圆圈，统一风格
  - 八字 → ☯、运势 → ☀、姻缘 → ⚭、易经 → ☰、取名 → ☲、风水 → ☴
- 宽字间距、细字重用于标题，营造呼吸感

### 字体

- 网页端：系统默认无衬线字体（`-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`）
- 小程序：微信默认字体

## 五、技术架构

### 5.1 技术栈

| 层 | 选型 | 说明 |
|----|------|------|
| 前端框架 | Next.js 14+ (App Router) | 全栈框架，SSR/SSG 友好 |
| UI | Tailwind CSS | 原子化样式，快速开发 |
| 后端 API | Next.js API Routes | 前后端一体，无需额外服务器 |
| 数据库 | PostgreSQL (Supabase) | 免费额度够起步，自带 Auth |
| ORM | Prisma | 类型安全，迁移方便 |
| 支付 | 扫码付（起步）→ 微信支付 | 渐进式接入 |
| AI 生成 | DeepSeek API | 深度命理报告生成，极低成本 |
| 部署 | Vercel | 免费层，全球 CDN |
| 图标 | Unicode 原生符号 | 零依赖，风格统一 |

### 5.2 API 设计原则

- 所有后端 API 设计为 RESTful，返回 JSON
- 微信小程序直接复用同一套 API
- 认证使用 Supabase Auth（支持邮箱 + 微信 OAuth）

## 六、数据流

```
用户输入（出生日期/时辰/姓名等）
        │
        ▼
┌──────────────────┐
│   算法计算层      │  ← GitHub 开源库（lunar-js, bazi 等）
│ 农历/八字/六十四卦 │
└──────┬───────────┘
       │ 计算结果（结构化数据）
       ▼
┌──────────────────┐
│   内容生成层      │
│ 模板拼装 / AI生成  │
└──────┬───────────┘
       │ 展示内容
       ▼
┌──────────────────┐
│   展示层          │
│ SSR 渲染 → 浏览器  │
└──────────────────┘
```

## 七、开发阶段
