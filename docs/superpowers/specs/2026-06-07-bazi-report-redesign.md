# 八字深度报告页面重新设计

## 目标

- 专业：基于八字理论（格局法、用神、调候、十神关系）做有依据的分析
- 个性化：每个命盘的分析有差异性，不是模板填空
- 对得起付费用户：视觉呈现与婚姻报告同等级别，每章有专属可视化组件

## 方案：章节专业化渲染 + 扩展数据模型

### 数据模型扩展

将 `BaziDeepReport.sections` 从 `{title, summary, body, highlights}[]` 扩展为携带结构化数据的类型，每个章节有专属的 `data` 字段供前端可视化。

### 章节结构（7章）

1. 命盘格局详解 — 日主强弱仪表盘 + 格局类型 + 用神忌神调候标签 + 胎命身三宫卡
2. 十神组合分析 — 四柱十神流 + 频次条 + 关键组合识别
3. 事业与财运 — 行业方向 + 财富类型 + 风险边界
4. 感情与婚姻 — 配偶星定位 + 婚姻宫分析
5. 健康与性格 — 五行偏性映射 + 性格标签
6. 大运流年 — 拖拽滚动时间轴 + 当前/下一阶段对比
7. 综合总结 — 核心发现 + 近期/长期行动建议

### 去模板化策略

- 在 `buildBaziDeepReport` 中基于 `BaziResult` 做更多推导计算
- 格局类型基于月令十神判断
- 十神组合自动识别（食神制杀、伤官见官等）
- 用神忌神基于调候和五行平衡
- 每个结论标注推导依据
