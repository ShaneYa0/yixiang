# 六爻问象 · 视觉重设计

## 概述

重做易经占卜起卦页面的仪式动画，从现在"黑色视频背景+简单爻线"的方案，改为**宣纸水墨风格、纯代码动画**的沉浸式起卦体验。

## 设计方向

**传统复古** — 宣纸质感、水墨笔触、古书卷轴的感觉。不用视频，全部用 Canvas 2D + SVG + CSS 实现。

## 核心视觉元素（4个）

| 元素 | 作用 | 阶段 |
|------|------|------|
| 宣纸底 | 背景氛围，米白微黄带纤维肌理 | 全阶段 |
| 墨晕流转 | 氛围铺垫，不规则扩散有浓淡变化 | 阶段 1-4 |
| 三钱落定 | 三枚铜钱依次落下，带旋转和墨迹拖尾 | 阶段 3 |
| 爻线成卦 | 毛笔一笔挥就，从下往上生长 | 阶段 3-5 |

## 五个阶段的视觉节奏

### 阶段 0 · 静心问卦（idle）
- 宣纸静止，中央隐约可见太极暗纹（纸背水印感）
- 底部微弱墨晕缓慢流转
- 六个爻位极淡虚线框，暗示六层结构
- 等待用户点击

### 阶段 1 · 一念初定（breath）
- 画面中央一滴浓墨垂直落入
- 触底后缓缓晕开，第一团墨晕形成
- 第一层爻位虚线微微加深

### 阶段 2 · 阴阳成旋（cloud）
- 墨晕不规则圆形扩散，浓淡变化，边缘毛刺/飞白
- 第二层爻位虚线变实

### 阶段 3 · 三钱定爻（coins）
- 三枚铜钱依次从上方落下，带旋转
- 第一枚落→停留→第二枚落→停留→第三枚落
- 落点处墨迹迸溅微粒
- 钱面：阳面（字面）、阴面（图案面），平面印章感
- 落定后生成第一根爻线，4-6层爻位依次点亮

### 阶段 4 · 六爻成象（seal）
- 剩余爻线逐层从墨中生长出来
- 每根线：毛笔笔触风格，起笔略顿、行笔有枯润、收笔自然
- 从下往上排列：初爻在下，上爻在上
- 最后一根爻线完成后整体微光一瞬

### 阶段 5 · 卦象已成（done）
- 六爻完整呈现
- 下方浮现卦名（大字号）
- 若有变爻，变爻处以朱砂色标注
- 墨晕凝固，不再流动

## 铜钱设计

- 圆形直径 ~24px（画面比例）
- 深褐色外圆，方孔内有细纹
- 平面"印章"感，非 3D 渲染
- 落地时有极短墨迹拖影

## 爻线笔触

- SVG path / Catmull-Rom 曲线模拟毛笔一笔挥出
- 中间粗两端略细，边缘有轻微锯齿（宣纸吃墨感）
- 阳爻（—）：一笔贯穿
- 阴爻（- -）：中间断开留白
- 使用 stroke-dashoffset 动画做"生长"效果

## 技术方案

- **Canvas 2D**：墨晕扩散、铜钱动画、墨迹粒子
- **SVG path**：爻线绘制与生长动画
- **CSS 背景**：宣纸纹理（多层渐变+噪点叠加）
- **requestAnimationFrame**：动画主循环
- **无外部依赖**：不引入动画库，纯手写

## 响应式
- 画面比例：桌面 4:3 → 移动端 3:2
- 铜钱和爻线等比缩放
- 文字输入和按钮保持 ≥16px 可点击区域
